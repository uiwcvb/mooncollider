为什么选择 MoonCollider？

从向量数学到刚体世界，一站式覆盖 2D 物理游戏的全部碰撞与模拟需求

📐

向量和形状

Vec2 全套运算；AABB / Circle / 凸多边形，支持凸包、正多边形、矩形构造器，Shape 枚举统一分发。

💥

SAT 窄相碰撞

全部 9 种形状对，分离轴定理求穿透法线和深度，2 点接触流形，含圆心在多边形内的退化处理。

🌀

GJK + EPA

通用凸形碰撞：GJK 迭代求单纯形，EPA 展开求最近穿透深度，任意 support 函数均可接入。

🔦

射线检测

Ray vs AABB / Circle / Polygon，返回参数化 t、命中点和表面法线，支持内外两面命中。

🗺️

四种宽相

GridHash 均匀网格、QuadTree 松散四叉树、SweepAndPrune 排序扫描、AABBTree 动态树（含增删改）。

⚙️

刚体求解器

顺序冲量法、Baumgarte 位置修正、库仑摩擦、2 点接触，velocity iterations 可调。

🔗

关节约束

Distance 距离关节、Revolute 旋转铰链、Weld 焊接关节，软 Baumgarte 偏置稳定收敛。

🎯

CCD 连续碰撞

扫描体求交，防止高速物体穿透薄墙，circle/AABB sweep 两种实现，可开关对比。

🛡️

对抗性 fuzz + 确定性

6 组 fuzzing（随机形状暴雨 / 退化几何 / NaN / 能量发散 / 穿模）+ 跨平台浮点快照，xorshift32 可复现。

核心架构

五层分层设计，每层可独立使用，也可从形状一路走到完整物理世界

World 求解层 — world.mbt
World::step 顺序冲量求解：力积分 → 宽相 → 窄相 → 接触约束 → 摩擦 → Baumgarte 位置修正
关节层 — joints.mbt
Distance / Revolute / Weld 三种约束，软 Baumgarte 偏置，与接触约束同帧求解
宽相层 — broadphase.mbt / broadphase_sap.mbt
GridHash / QuadTree / SweepAndPrune / AABBTree，候选对生成，O(n) 到 O(n log n)
窄相层 — narrowphase.mbt / gjk.mbt / raycast.mbt / ccd.mbt
SAT 9 对 + GJK/EPA 通用凸形 + 射线 + CCD 扫描，输出 Manifold（法线 / 深度 / 接触点）
基础层 — vec2.mbt / shape.mbt / body.mbt
Vec2 数学、AABB/Circle/Polygon 几何、RigidBody 刚体定义（质量 / 转动惯量 / 恢复系数 / 摩擦 / 阻尼）

快速开始

三行命令安装，十行代码跑起一个物理世界

bash安装
# 在你的 MoonBit 项目里
moon add bubuki/mooncollider
moonbit弹跳球：动态圆 + 静态地板
fn main {
  let world = @mooncollider.World::new()
  // 动态圆球
  let _ = world.add_body(
    @mooncollider.BodyDef::dynamic(
      @mooncollider.Vec2::new(0.0, 10.0),
      @mooncollider.Shape::from_circle(
        @mooncollider.Circle::new(@mooncollider.Vec2::zero(), 0.5)),
      1.0, 0.8))
  // 静态地板
  let _ = world.add_body(
    @mooncollider.BodyDef::static_(
      @mooncollider.Vec2::new(0.0, 0.0),
      @mooncollider.Shape::from_aabb(
        @mooncollider.AABB::from_min_max(
          @mooncollider.Vec2::new(-10.0, -1.0),
          @mooncollider.Vec2::new(10.0, 0.0)))))
  // 步进仿真
  for i = 0; i < 200; i = i + 1 {
    world.step(0.016)
  }
}
moonbit单独碰撞检测（不用 World）
let a = @mooncollider.Shape::from_aabb(
