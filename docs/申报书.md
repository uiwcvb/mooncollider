# MoonCollider 项目申报书

## 1. 项目名称和仓库

- **项目名称**：MoonCollider
- **GitLink 仓库**：https://gitlink.org.cn/bubuki/mooncollider
- **GitHub 仓库**：https://github.com/uiwcvb/mooncollider
- **mooncakes 包**：`moon add uiwcvb/mooncollider`

## 2. 项目简介

MoonCollider 是面向 MoonBit 的 2D 游戏物理与碰撞检测库，提供 SAT 窄相、GJK+EPA、四种宽相加速结构、顺序冲量刚体求解器、关节约束和 CCD。纯逻辑无渲染，跑遍 wasm-gc/js/native。约 5400 行 MoonBit 代码，61 个测试。

## 3. 项目方向与适用场景

2D 游戏物理引擎，适用于平台跳跃、物理益智、仿真教学、关卡编辑器碰撞检查。填补 mooncakes.io 物理引擎空白。

## 4. 核心功能

全部完成：Vec2/AABB/Circle/Polygon 几何、9 种碰撞对 SAT、GJK+EPA、射线检测、GridHash/QuadTree/SAP/AABBTree 四种宽相、顺序冲量求解器（Baumgarte+摩擦+2点接触）、Distance/Revolute/Weld 关节、CCD、body removal、交互式网页沙盒。

## 5. 原创性声明

原创项目，全部代码从零用 MoonBit 编写。参考 Box2D（MIT, https://github.com/erincatto/box2d）求解器架构和 Real-Time Collision Detection（ISBN 978-1558607323）SAT/GJK/EPA 理论，未移植源代码。Apache-2.0 协议。
