# MoonCollider 项目申报书

## 1. 项目名称和仓库地址

- **项目名称**：MoonCollider
- **GitLink 仓库**：https://gitlink.org.cn/bubuki/mooncollider
- **GitHub 仓库**：https://github.com/uiwcvb/mooncollider
- **mooncakes 包**：`moon add bubuki/mooncollider`

## 2. 项目简介

MoonCollider 是面向 MoonBit 语言生态的 2D 游戏物理与碰撞检测库。它提供了从向量数学到刚体仿真的完整物理管线：SAT 分离轴窄相碰撞、GJK+EPA 通用凸形检测、四种宽相加速结构（均匀网格/四叉树/排序扫描/动态 AABB 树）、顺序冲量刚体求解器（Baumgarte 位置修正 + 库仑摩擦 + 2 点接触流形）、三种关节约束（距离/旋转/焊接）、CCD 连续碰撞检测，以及射线求交。全部为纯逻辑实现，无渲染依赖，可运行于 wasm-gc / js / native 三个后端。

项目规模约 5000 行 MoonBit 代码，含 56 个测试（黑盒 + 白盒 + 对抗性 fuzzing + 确定性快照），配套 7 个可运行 demo 和 1 个由真实引擎驱动的交互式网页沙盒。

## 3. 项目方向与适用场景

- **方向**：游戏物理引擎 / 碰撞检测中间件
- **适用场景**：
  - 2D 游戏：平台跳跃、物理益智、弹球消消乐
  - 仿真教学：刚体动力学演示、碰撞响应可视化
  - 工具链：关卡编辑器中的物体放置碰撞检查、射线拾取
  - 生态补全：MoonBit 目前 mooncakes.io 上无物理引擎包，本项目填补空白

## 5. 原创性声明

本项目为**原创项目**。碰撞检测算法参考了 Box2D（Erin Catto）和 Real-Time Collision Detection（Christer Ericson）的经典理论，但全部代码从零用 MoonBit 编写，未移植任何现有物理引擎的源代码。项目采用 Apache-2.0 开源协议。
